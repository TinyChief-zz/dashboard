# Dashboard
Easy to add to every site. Just add files from `/dist` to `/dashboard` of your site to access this from `www.yourdomain.com/dashboard`

## Implemented features
* **Messages**
* **Chart**
* **Notes**
* **Map**
* **Chat**
* **News**

See live [here](https://my-dashboard.netlify.com)
